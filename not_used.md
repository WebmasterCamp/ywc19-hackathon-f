# Group F

## Tech Stacks

- Frontend:
  - Next.js static build

- Backend:
  - Nest.js
  - postgresql

## Build Script

- Frontend & Backend:

  1. Go to `frontend` directory

     ```bash
     cd frontend
     ```

  2. Copy `.env.template` to `.env` && Config the key

     ```bash
     cp .env.template .env
     ```

  3. Install

     ```bash
     npm install
     ```

  4. Build

     ```bash
     npm run build
     ```

  5. Start server

     ```bash
     npm start
     ```
